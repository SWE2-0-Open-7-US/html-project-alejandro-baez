Projects

Grace-Shopper: Book-Worm

The Grace-Shopper project was a group e-commerce website of our choosing that incorporated backend and frontend elements. This culminated in a fullstack application that used Node, Express, React, PostgreSQL, and Sequelize. One of my main contributions to the project was working with the different book API's to seed our database and inject live data into our program. Ultimately, the Google Books API was used to retrieve a book's author, ISBN, price, and genre

Capstone: Pictionary

The Capstone project was a group project of our choosing. We decided to create a fullstack Pictionary style game using Node, Express, PostgreSQL, Sequelize, and Socket.js. One of my main contributions to the project was working with socket.js to establish a real-time connection between two players. Also used socket.js to generate rooms and codes so multiple games could occer at the same time.
